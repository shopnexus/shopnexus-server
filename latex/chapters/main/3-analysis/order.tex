\subsection{Thành phần nghiệp vụ}

Module phân rã thành: \textbf{Cart} (GetCart, UpdateCart, ClearCart), \textbf{Checkout} (BuyerCheckout, ListBuyerPending, CancelBuyerPending), \textbf{Incoming} (ListSellerPending, QuoteTransport, ConfirmSellerPending, RejectSellerPending), \textbf{Order} (GetBuyerOrder, GetSellerOrder, ListConfirmed, ConfirmPayment), \textbf{Payment} (PayBuyerOrders), \textbf{Dashboard} (Stats, TimeSeries, PendingActions, TopProducts).

\section{Sơ đồ chuyển trạng thái đối tượng}
% ============================================================

Phần này trình bày sơ đồ chuyển trạng thái cho các đối tượng nghiệp vụ cốt lõi trong module đặt hàng: mục đơn hàng (item), đơn hàng (order), thanh toán (payment) và vận chuyển (transport). Các trạng thái và chuyển tiếp được suy ra trực tiếp từ enum trong cơ sở dữ liệu và interface \texttt{OrderBiz}.
