\section{Tổng quan hiện thực}
\label{sec:version-overview}

Phiên bản hiện tại của ShopNexus (\textbf{v1.0 Development}) đã hiện thực toàn bộ luồng nghiệp vụ cốt lõi của nền tảng thương mại điện tử, bao gồm: quản lý tài khoản và xác thực JWT, catalog sản phẩm SPU/SKU với tìm kiếm lai (Milvus + PostgreSQL), giỏ hàng và checkout với reserve inventory, thanh toán qua VNPay và COD, quản lý đơn hàng cho buyer/seller, hoàn trả và tranh chấp, khuyến mãi tự động, chat real-time, phân tích hành vi người dùng, và dashboard thống kê cho seller.

\subsection{Triển khai}
\label{subsec:deployment}

Môi trường phát triển sử dụng \textbf{Docker Compose} (\texttt{deployment/docker-compose.yml}) để quản lý hạ tầng gồm: PostgreSQL, Redis, Restate cluster 3 node (replication factor 2, snapshot lưu trên MinIO), Milvus standalone với etcd, và MinIO cho object storage. Tất cả container nằm trong Docker network chung. Go server chạy trên host với hot-reload qua \texttt{air} (\texttt{make dev}), kết nối các service qua port mapping. Quy trình khởi tạo: \texttt{docker compose up} $\rightarrow$ \texttt{make migrate} $\rightarrow$ \texttt{make seed} $\rightarrow$ \texttt{make dev} $\rightarrow$ \texttt{make register}.
